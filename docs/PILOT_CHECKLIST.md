## Instalación y actualización

- [ ] La APK se instala sin desinstalar la versión anterior.
- [ ] La aplicación conserva inventario, técnicos y movimientos.
- [ ] La aplicación conserva usuarios y PIN.
- [ ] La aplicación conserva fotografías.
- [ ] El diagnóstico muestra SQLite operativo.
- [ ] El número de herramientas coincide antes y después.
- [ ] El número de movimientos coincide antes y después.

## Acceso y seguridad

- [ ] Se puede crear el administrador inicial.
- [ ] El PIN correcto desbloquea la aplicación.
- [ ] El PIN incorrecto es rechazado.
- [ ] Cinco intentos fallidos bloquean temporalmente el usuario.
- [ ] El bloqueo automático se activa por inactividad.
- [ ] Administrador, Almacén y Técnico muestran acciones diferentes.
- [ ] El Técnico no puede registrar ni modificar datos.
- [ ] El operador correcto aparece en cada movimiento.

## Cámara y QR

- [ ] Android solicita permiso de cámara.
- [ ] El lector QR abre correctamente.
- [ ] Se escanea un QR de técnico.
- [ ] Se escanea un QR de herramienta.
- [ ] Se detecta un QR no perteneciente a ISIVOLT.
- [ ] La entrada manual permite continuar cuando la cámara no está disponible.
- [ ] La etiqueta impresa se lee a una distancia normal.
- [ ] El lector funciona con iluminación baja y media.

## Entregas y devoluciones

- [ ] Entrega individual correcta.
- [ ] Entrega múltiple correcta.
- [ ] Devolución correcta.
- [ ] Devolución para revisión exige observación.
- [ ] Devolución averiada exige observación.
- [ ] No se puede entregar una herramienta ya prestada.
- [ ] No se puede entregar una herramienta bloqueada por mantenimiento.
- [ ] Una reserva solo puede entregarse al técnico previsto.
- [ ] Sonido y vibración funcionan y pueden desactivarse.
- [ ] Las animaciones no dificultan el trabajo.

## Fotografías

- [ ] Se puede hacer una fotografía.
- [ ] Se puede seleccionar una fotografía de galería.
- [ ] La fotografía continúa después de cerrar y abrir.
- [ ] La fotografía continúa después de actualizar la APK.
- [ ] La herramienta sin foto muestra un marcador correcto.

## Gestión

- [ ] Crear herramienta.
- [ ] Editar herramienta.
- [ ] Código duplicado rechazado.
- [ ] Crear y editar técnico.
- [ ] Técnico con material prestado no puede desactivarse.
- [ ] Crear accesorios.
- [ ] Marcar accesorio faltante o dañado.
- [ ] Abrir expediente de mantenimiento.
- [ ] Cerrar expediente de mantenimiento.
- [ ] Configurar revisión y calibración.
- [ ] Las alertas muestran vencimientos correctamente.
- [ ] Importar una plantilla Excel.

## Informes y recuperación

- [ ] Generar Excel operativo.
- [ ] Generar Excel de gestión.
- [ ] Compartir los archivos.
- [ ] Crear copia JSON.
- [ ] Restaurar copia en un entorno de prueba.
- [ ] La restauración recupera accesorios y mantenimiento.
- [ ] La restauración conserva la trazabilidad.

## Rectificaciones

- [ ] El administrador puede rectificar un movimiento.
- [ ] El movimiento original permanece visible.
- [ ] Se crea un movimiento de ajuste nuevo.
- [ ] No es posible rectificar dos veces el mismo movimiento.
- [ ] La auditoría registra la rectificación.

## Prueba sin conexión

- [ ] Activar modo avión.
- [ ] Abrir la aplicación.
- [ ] Desbloquear con PIN.
- [ ] Consultar inventario.
- [ ] Registrar entrega y devolución.
- [ ] Añadir una fotografía.
- [ ] Cerrar y volver a abrir.
- [ ] Confirmar que los datos continúan.
