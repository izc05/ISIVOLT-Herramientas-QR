# Prueba física del escáner web RC30

## Preparación

1. Abrir `https://izc05.github.io/ISIVOLT-Herramientas-QR/` en Chrome Android.
2. Actualizar la página y confirmar que aparece **Modo web RC30**.
3. Pulsar el botón principal de escaneo.
4. Elegir préstamo o devolución y el orden de identificación.

## Primera apertura de cámara

1. Pulsar **Cámara técnico** o **Cámara herramienta**.
2. Leer el aviso de privacidad.
3. Pulsar **Activar cámara**.
4. Autorizar el permiso del navegador.
5. Confirmar que se abre preferentemente la cámara trasera.

## QR de herramienta

1. Colocar el QR dentro del marco.
2. Mantener el teléfono estable y con luz suficiente.
3. Confirmar que el visor se cierra automáticamente al detectar.
4. Verificar que la herramienta correcta queda añadida una sola vez.
5. Repetir en préstamo y devolución.

## Tarjeta corporativa de técnico

1. Probar el código horizontal.
2. Probar el código vertical girando la tarjeta o el teléfono.
3. Confirmar que el técnico asociado se identifica correctamente.
4. Probar una tarjeta no registrada y verificar el mensaje de aviso.

## Cancelación y seguridad

1. Abrir el visor y pulsar **Cancelar**.
2. Volver a abrirlo y comprobar que la cámara está disponible.
3. Cerrar la pestaña mientras el visor está activo.
4. Volver a abrir la web y comprobar que la cámara no quedó bloqueada.
5. Denegar temporalmente el permiso y revisar que aparece un mensaje recuperable.

## Criterio de resultado

Registrar para cada prueba:

- Dispositivo y versión de Chrome.
- Código probado.
- Resultado correcto o fallo.
- Tiempo aproximado de lectura.
- Captura del mensaje cuando exista un error.

El bloque se considera validado cuando funcionan un QR de herramienta y las orientaciones horizontal y vertical de la tarjeta corporativa sin dejar ningún stream de cámara activo.
